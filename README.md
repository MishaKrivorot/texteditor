# Text Editor (WinForms / WPF)

Простий текстовий редактор, розроблений у рамках лабораторної роботи з використанням Windows Forms та Windows Presentation Foundation (.NET).

## Функціонал

- Створення нового файлу
- Відкриття існуючого файлу (.txt)
- Редагування тексту
- Збереження файлу
- Збереження файлу під новим іменем (Save As)
- Перевірка незбережених змін перед виходом

## Технології

- C#
- .NET (WinForms / WPF)
- System.IO для роботи з файлами

## Запуск

1. Відкрити рішення у Visual Studio  
2. Обрати потрібний проєкт (WinForms або WPF)  
3. Запустити програму (F5)
